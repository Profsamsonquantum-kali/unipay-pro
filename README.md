# QuantumPay Backend API

QuantumPay V20 Ultimate Pro Backend - A comprehensive financial platform API.

## Features
- User authentication (register/login)
- Multi-currency accounts
- Crypto wallet integration
- Mobile money support (M-Pesa, MTN, Orange, Airtel)
- Virtual cards
- Loans and investments
- Real-time WebSocket updates

## Tech Stack
- Node.js / Express
- MongoDB with Mongoose
- JWT Authentication
- Socket.io for real-time updates
- Web3 for crypto integration
